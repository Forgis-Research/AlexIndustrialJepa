% V16b vs V2 label-efficiency figure
% Shows that V16b (bidi + VICReg) fails catastrophically at low labels
% vs V2 (causal), strengthening the causal inductive bias story.
%
% Data source: experiments/v16/phase8_label_efficiency.json (V16b)
%                + v14 phase results for V2 (from RESULTS.md / paper table)
% V2 E2E: 14.23 / 14.93 / 16.54 / 18.66 / 25.33 at 100/50/20/10/5%
% V16b E2E: 15.36 / 22.43 / 27.01 / 29.27 / 40.65 (with std 1.07 / 6.28 / 5.28 / 1.06 / 10.18)
